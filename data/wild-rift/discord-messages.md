## 🎯 Callouts de Objetivos (3 mensajes)

### Callout 1: Dragón

```
🐉 DRAGÓN EN 1:30 — TODOS PREPARARSE 🔥

📍 JUNGLA: Empieza a moverse hacia dragón YA
🗡️ BARON: Empuja lane y muévete al río
🔮 MID: Empuja y baja al río por la zona de pixeles
🐉 DRAGON: Empuja y muévete con el support
🛡️ SUPPORT: Wards en la entrada del dragón AHORA

⚠️ SI ESTÁ MUERTO O BACKED: PUSH Y FORZAR DRAGÓN INMEDIATAMENTE
```

### Callout 2: Barón

```
🦅 BARÓN — ENEMIGO FUERA DEL MAPA 🏰

⚡ TODOS MUÉVANSE A BARÓN — El enemy no puede contestar
🗡️ BARON: Empuja TOP y teleport/barrier a barón
🐺 JUNGLA: Controla la entrada, smite ready
🔮 MID: Clear la visión antes de empezar
🐉 DRAGON: Posición en la entrada de barón
🛡️ SUPPORT: ORACLE + CONTROL WARD en barón pit

🎯 Si el enemy aparece: PELEAMOS O NOS VAMOS — No empezamos barón sin seguridad
```

### Callout 3: Inhibidor

```
🏰 PUSH MID — INHIBIDOR ABIERTO 💥

⚡ Todos a mid lane — PUSH PUSH PUSH
🗡️ BARON: Tank en el frente, clear de minions
🐺 JUNGLA: Smite los minions rápido
🔮 MID: ULTI en la torre si es posible
🐉 DRAGON: Daño a la torre desde atrás
🛡️ SUPPORT: PEEL y heals para mantener el push

🏆 SI CAE EL INHIB: Recuerden — NO SE QUEDEN EN BASE, vuelvan a hacer barón o farm
```

## 📢 Templates de Shotcall (3 mensajes)

### Shotcall 1: Early Game

```
📢 SHOTCALL EARLY GAME — MINUTO 3-5 ⏰

🐺 JUNGLER: ¿Dónde empezaste? Comparte con el equipo
📍 Si empezaste BOT: Gank TOP a los 3:15-3:30
📍 Si empezaste TOP: Gank BOT a los 3:15-3:30

🎯 Dragón spawnea en minuto 4:00
🐉 Si hacemos kill en bot/top: VAMOS A DRAGÓN
⚠️ Si no hay kills: NO FORCEN — esperen al segundo dragón

📌 Todos: Pongan wards en el río ANTES del minuto 3
```

### Shotcall 2: Mid Game

```
📢 SHOTCALL MID GAME — 10-20 MINUTOS 🎯

🐉 DRAGÓN: ¿Cuántos tenemos? [X]/4
🔥 Si tenemos 3: PRIORIDAD ABSOLUTA — eso es DRAGON SOUL potential
🕊️ HERALDO: Si no lo usamos, el enemy lo va a usar contra nosotros

🏛️ TORRES: Si tomamos 2 torres exteriores, el map se abre para INVADIR
🦅 BARÓN: NO barón antes de 20 minutos a menos que TENGAMOS ventaja masiva

⚡ REGLA: PUSH ➡️ OBJETIVO ➡️ RECALL ➡️ REPETIR
🚫 NUNCA se queden sin hacer nada — siempre tenga un objetivo
```

### Shotcall 3: Late Game

```
📢 SHOTCALL LATE GAME — 20+ MINUTOS 🏰

💀 ONE PICK = GAME OVER — Si alguien muere, podría perder el game entero

🛡️ REGLA #1: Stay together — NO SE SEPAREN
👁️ REGLA #2: Wards en la jungle enemiga — necesitamos VERlos
🦅 REGLA #3: Si el enemy hace barón, evaluamos: ¿Podemos steal o contestar?

⚡ SI TOMAMOS INHIB: PUSH las otras lanes ANTES de volver a pelear
🎯 SI ESTAMOS DETRÁS: DEFENDAN y esperen un error del enemy
🏆 SI ESTAMOS ADELANTE: NO THROW — objetivamente es lo peor que pueden hacer

🎪 EN LATE GAME, UNA SOLA PELEA GANA O PIERDE EL JUEGO — Elijan sabiamente ⚔️
```
